 Banking & Finance System (MySQL Project)
## Overview##
This project simulates a real-world banking system using MySQL. It includes customer management, account handling, transactions, loan processing, EMI calculation, fraud detection, and audit logging.

 ##Database Design##
The system consists of 6 core tables:
1)customers - Stores customer details
2)accounts - Stores account information
3)transactions - Records all financial transactions
4)loans - Manages loan details
5)emi_schedule - Tracks EMI payments
6)audit_log - Logs all system activities

 How to Run the Project

Run the following commands in MySQL Workbench:

SOURCE schema.sql;
SOURCE data.sql;
SOURCE procedures.sql;
SOURCE triggers.sql;
SOURCE queries.sql;
SOURCE views.sql;

Key Concepts Used
1)ACID Transactions
2)Foreign Keys & Constraints
3)Window Functions
4)CTE (Common Table Expressions)
5)Triggers & Automation
